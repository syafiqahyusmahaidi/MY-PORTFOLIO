Musism Web Application

Homepage

After logging in, the user will be directed to the main menu, which will display the homepage of this web application. On the topbar menu, it will display some function that can be clicked on such as Home, Product List, About Us, search box, the user name, and logout.

Product List Page

If user click on “Product List” navigation bar menu on top of the page, it will display all the products that available. It also have pagination menu on top of the product lists to ease the user to go to next page. In every product, have a “View Product” button. It will lead to the specific websites of the product to ease user buying and see the details of product.

Filter Brands

The filter bar on the left of the page which have list og brands and category. When user click on one of the menu, it will display according to the menu. It will redirect to search page.

Search Result

the search result after user input keyword of their desire musical instruments in the search box at the navigation bar. Users can input any keyword that related to the products. The search engine will display all products that related to the keyword.

My Account Page

My Account page which will display the details of the user such as the username, email and password. It also have two button which edit and back. Edit button will redirect to edit page which user can edit their details. Back button will redirect to the homepage.

Edit Profile

User can edit their details such as name, email and password. There are two button which are submit and back. The function of submit button is when user already fill the form to change the details, it will save the new details. Back button will redirect to My Account page.

Admin's product list

This page will appear after admin click on ‘Products’ button. It will display all the products that available in the database. It displays the product details such as title, category, vendor and price. On the right of each product, there are two button, edit and delete.
